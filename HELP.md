## Motu AVB

This module connects to a Motu AVB audio interface.

### Configuration
* Type in the device IP address.
* The device will communicate over port 37297 by default.

### Available actions
* Set Channel mute
* Set Main mute
* Set Group mute
* Set monitor mute
* Channel fader level
* Main fader level
* Group fader level
* Monitor fader level
* Set Solo on Channel
* Set Solo on Group
